Puzzle Checklist

  1. Check the HTML Source for puzzle
  2. Does the title of the puzzle help at all?
  3. If multiples, is there anything in common? Try searching
  4. Can it be anagrammed?
  5. First/Later letters of each thing spell anything
  6. Can the elements be split into groups?
  7. Can the groups be morse/binary encoded? Check 1st, 9th, 17th etc for 0/1/something in common
  8. How many elements are there? If 26 - are they letters? If multiple of 8 is it binary?
  9. Can this be converted to or from braille?
  10. Is this some kind of ASCII art for a word?
  11. Can multiple characters be treated as one? e.g. those with a common prefix or suffix?
  12. Look at the letters on the a keyboard, is there a pattern here?
  13. Is the number of characters square? Can they be arranged in a grid?
  14. Can the picture be a simple word
  15. If there are multiple pictures, is there a theme commonality?
  16. Check Exif/Size
  17. Could it be a flag? Enter some search terms to find flags
  18. Run through strings/file, look at results
  19. Check file names
  20. What times do events happen or how long do they take
  21. Check file names
  22. Run through strings/file, look at results
  23. Check channels match
  24. Reverse
